NewAge css framework

NewAge is a small css framework based on principles of material design.

v0.1.3

Typography

By default using Roboto font. But you can use another, only change property font-family:


html {
	font-family: Arial, Trebuchet, "...", sans-serif;
}

For different weight of Roboto font, use property font-weight with values: 100, 300, 400, 500, 700. The default value is 400.
